Why Atom?

🐍 Python-Like Syntax

Clean, readable, and intuitive. If you know Python, you already know Atom.

⚡ Native Performance

Compiles to machine code via C. Fast startup, small binaries, no runtime overhead.

🌐 WebAssembly Ready

Deploy anywhere: browsers, edge computing, serverless, or embedded devices.

🤖 AI Built-in

TinyLLM included. Support for LLaMA.cpp and ONNX runtime.

📦 Package Manager

Simple dependency management. `atom add http` installs packages instantly.

🌍 Web Server

Built-in HTTP server with routing, middleware, and static file support.

Install Atom

Clone the repository and start coding in seconds

git clone https://github.com/Mharshith548/atom.git

Then run:

cd atom && python atom.py repl

No dependencies required! Works on Python 3.8+

chatbot.atom 
# AI Chatbot
import ai

let model = ai.load("tinyllm")

while true:
    let user = input("You: ")
    if user == "quit":
        break
    print("Bot: " + model.ask(user))
server.atom 
# Web Server
import http

let server = http.Server(8080)

server.route("/", fn(req):
    return {"status": 200, "body": "Hello!"}
)

server.start()
